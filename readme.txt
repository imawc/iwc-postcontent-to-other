== Description ==

You can display the content of a posted article on a separate fixed page or in a posted article.

== Installation ==

1. Search `IWC PostContent to Other` on [WordPress Plugins](https://wordpress.org/plugins/) or Plugin install on your admin.
2. Install and Activate the plugin through the 'Plugins' menu in WordPress.
3. By pasting the specified shortcode, you can display the content of the article.

shortcode example
`[iwc-postby id="postid"]`
